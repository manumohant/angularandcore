1. download and install node and vs code from this link
https://nodejs.org/en/download/
https://code.visualstudio.com/download

2. Open terminal and type 
npm install -g @angular/cli


3. Extract the zip file and angularapp.zip open the folder and goto the root directory of angular app where you can see package.json. 
   either open the folder location in terminal or open the entire project in vs code
   type the below command in the windows terminal or vs code terminal
npm install
4. after finishing npm install(which will take few minuts ) run below command
ng serve --open
